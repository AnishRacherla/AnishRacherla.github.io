Intelligent Parking System

  • This Intelligent Parking System is built using an ESP32 microcontroller, RFID reader, Firebase Realtime Database, and IR sensors.
  • When an authorized RFID tag is scanned, the gate opens, and the system checks if the car is parked in the correct spot using IR sensors.
  • If the car is parked correctly, it confirms the placement; otherwise, it alerts about wrong parking or if no parking occurred.
  • The project demonstrates real-time authentication, sensor integration, and cloud communication.
